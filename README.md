# Credit Risk Data Pipeline

This project demonstrates an end-to-end data pipeline for credit application analysis.

## 🔧 Technologies
- PostgreSQL
- Pentaho Data Integration (ETL)
- Power BI
- Airflow 

## 📊 Project Scope
- Data modeling (customers, applications, decisions)
- ETL pipeline design and implementation
- KPI calculations (approval rate, loan volume, etc.)
- Interactive dashboard development
